# Tesi-2026

Questo repository contiene tutto il materiale, i dati e il codice sviluppato per il progetto di tesi magistrale in **Scienze Statistiche** presso **Sapienza Università di Roma**.

## Obiettivo del Progetto

Il progetto si concentra sullo sviluppo di un approccio  basato sui grafi (Knowledge Graphs) e architetture **GraphRAG** per tracciare la validità storica, l'evoluzione e le modifiche delle norme giuridiche italiane. 
Sfruttando l'integrazione tra database a grafo, vector embeddings e Large Language Models (LLM), il sistema permette di interrogare dinamicamente il corpus normativo tenendo conto delle finestre temporali di vigenza.

## Tecnologie Utilizzate

*   **Linguaggi:** R, Python
*   **Database & Ricerca:** Neo4j (Knowledge Graph), OpenSearch (Ricerca ibrida e vettoriale)
*   **AI & NLP:** Ollama, Mistral, Nomic-embed-text
*   **Origine Dati:** API Open Data di Normattiva
